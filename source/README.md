# 수학 교육 통합 플랫폼

교사와 학생을 연결하는 **AI 기반 수학 학습 생태계** · 대상: 고등학교(인공지능 수학)

> 본 산출물은 별도 설치 없이 **`index.html`을 브라우저로 열면** 바로 동작합니다(데모 모드).

---

## 1. 실행 방법

### 가장 간단한 방법 (데모 모드 — 심사용)
1. `program/index.html` 을 크롬/엣지/사파리로 엽니다.
2. 데이터는 브라우저 localStorage에 저장되며, **샘플 학생 5명(1학년 7반)** 이 미리 들어 있어 바로 둘러볼 수 있습니다.
3. ※ 카메라·일부 실시간(다기기) 기능은 보안상 HTTPS(아래 Firebase 배포)에서 동작합니다.

### 실서비스 모드 (Firebase 연결)
1. Firebase 콘솔에서 프로젝트 생성 → Authentication / Realtime Database / Storage 활성화
2. `js/firebase.js` 의 `firebaseConfig` 값을 콘솔 값으로 교체
3. `firebase deploy` (또는 정적 호스팅) → HTTPS 환경에서 전체 기능 동작

### 테스트 계정
| 역할 | 아이디 | 비밀번호 |
|------|--------|----------|
| 관리자 | `Admin01` | `admin123` |
| 교사 | `teacher` | `teacher123` |
| 학생(샘플) | `10701` ~ `10705` | 아이디와 동일 |

> 학생 아이디 = 학년반번호 (예: 1학년 7반 1번 → `10701`)

### AI 기능 사용
- AI 챗봇·서술형 채점·PDF 문항 추출·관찰기록 등은 **교사가 설정(⚙️)에서 Claude API 키를 직접 입력**해야 동작합니다.
- 키는 **소스에 포함되지 않으며** 교사 계정에만 저장됩니다.

## 2. 주요 기능

1. **스케줄러** — 달력·일정·투두, 교사 → 학생 일정 공유
2. **메신저** — 실시간 1:1 채팅, 사진·카메라, 알림 배지/푸시
3. **학습실**
   - 교과수업 — 슬라이드(개념/수업노트/유튜브/웹임베드/평가/자기성찰일지), 실시간 발표 동기화, 위험신호 조기경보
   - 협업의 장 — 실시간 협업 화이트보드 · **배움 대전**(스피드 퀴즈 게이미피케이션)
   - 과제방 — 등록·알림·제출·AI 사진 분석·기한 경고
   - 자료방 — 교사 자료 공유
4. **질문방** — 수학자 캐릭터 소크라테스식 AI 멘토, 공개 질문, 포인트·뱃지
5. **기타** — 수학 그림 작성기 · 바이브코딩(거북이 그래픽) · 다국어(한/영/중/베)
6. **포트폴리오** — 5대 핵심역량 레이더·뱃지·AI 관찰기록·PDF 출력

### 평가 핵심
- 9개 문항 유형(객관식 5지선다·다중·참거짓·단답·수식·그래프·순서·서술형·즉답통계)
- 자동 채점 + 서술형 AI 채점 → 5대 핵심역량 레이더 자동 반영
- **한 문제 = 한 페이지**(좌: 문제·그림·답 / 우: 태블릿 필기 노트)
- PDF 업로드 → AI가 문항 자동 추출(**그림·표·도표 자동 캡처 포함**)
- 캡처 이미지 붙여넣기(Ctrl+V)로 문항 제작
- 교사: 학생별 문항 정오(○/✕) 확인
