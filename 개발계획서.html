신규 추가 기능 — 메신저 위 첫 번째 메뉴

📅
핵심 개념: 교사가 일정을 공유하면 학생 달력에 자동 저장

교사가 스케줄러에 일정을 등록하고 "학생에게 보내기" 를 누르면, 선택한 반 전체 학생의 달력에 해당 일정이 Firebase를 통해 즉시 저장됩니다. 학생은 로그인하면 교사 일정을 바로 확인할 수 있습니다.

6-2. 달력 (Calendar)

교사용 달력

  • 월간·주간·일간 뷰 전환
  • 일정 등록: 제목·날짜·시간·색상·메모·대상반
  • "학생에게 보내기" 버튼 → 선택 반 즉시 공유
  • 반 전체 달력 모니터링 가능
  • 수업 일정·평가 일정·공지 색상 구분

학생용 달력

  • 교사 공유 일정: 파란색 (삭제 불가)
  • 과제 마감일: 주황색 (과제방 자동 연동)
  • 개인 일정: 회색 (본인만 열람·편집)
  • 일정 클릭 → 상세 내용 팝업
  • 미래 일정 알림 표시
📌
FullCalendar.js 사용 (MIT 라이선스, CDN)

드래그&드롭 일정 이동, 다중 뷰(월간/주간/일간), 반응형 지원. Firebase Realtime DB와 연동하여 교사가 일정 추가 시 학생 달력에 실시간 반영.

6-3. 투두리스트 (Todo List)

달력 우측 사이드패널

  • 날짜 클릭 → 해당 날짜의 할 일 목록 표시
  • 체크 완료 시 취소선 + Firebase 저장 (기기 간 동기화)
  • 드래그 앤 드롭으로 날짜 간 이동
  • 우선순위 설정 (높음·보통·낮음)

자동 연동

  • 과제방 과제 등록 → 학생 투두리스트 자동 추가 🟠
  • 교사 일정 공유 → 학생 투두리스트 자동 추가 🔵
  • 수업 시작 알림 → 당일 투두에 표시
  • 완료율 대시보드에 표시

핵심 기능 상세 설계

8-1. 인증 시스템

교사 계정

  • Firebase Auth 이메일 로그인
  • 관리자가 일반 교사 등록·관리
  • AI API 키 설정 메뉴

  • 엑셀 업로드 → 일괄 생성 (XLSX.js)

  • 초기 비밀번호 = 아이디, 첫 로그인 시 변경 강제
  • 교사가 비밀번호 초기화 가능
